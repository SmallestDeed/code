产品基本信息：

产品编码：系统自动生成，编码规则：P+序号（10位）。

产品分类：必填，调用三度后台产品分类目录树，下拉选项需要做级联

产品品牌：必填，调用三度后台品牌管理列表中的品牌数据

产品名称：必填，文本框，手动填写

产品规格：非必填，文本框，手动填写，单位：cm

产品其他基本字段：不同的产品分类，对应的基本信息不同，待整理








产品描述信息

产品系列：非必填，下拉选择，选项内容调用三度后台的产品系列

产品款式：非必填，下拉选择，选项内容调用三度后台的产品款式

产品风格：非必填，下拉选择，选项内容调用三度后台的产品风格

建议售价：非必填，文本框，手动输入价格

实际价格：非必填，文本框，手动输入价格

价格单位：填写了建议售价和实际价格，必须要选择价格单位，下拉选项，选项内容在三度后台配置

产品图片：非必填，选择本地图片上传，最多可上传10张图片，默认第一张为主缩略图，用户可设置其它图片为主缩略图。

产品描述：非必填，文本框，手动填写。

自定义属性：产品还有其他描述属性，可自定义描述属性字段。点击“添加更多自定义字段”，页面显示字段输入框和字段内容输入框。
